Apr 10, 2018

keep-alive

keep-alive 是 Vue 内置的组件，可以将 keep-alive 的子组件运行缓存，用于保存切换 (例如 v-if 的切换) 过程前的状态。

而 keep-alive 有两个 props 是 include 以及 exclude。前者指明需要缓存的组件的名字。而后者则指明不需要缓存的组件的名字。
